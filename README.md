# Web Frontend

[![CircleCI](https://circleci.com/gh/labsoft-2018/web-frontend/tree/master.svg?style=svg)](https://circleci.com/gh/labsoft-2018/web-frontend/tree/master)

## Running tests (snapshot & unit)
```
yarn run test
```

*in watch mode*:
```
yarn run test:watch
```

## Bundling
```
yarn run build
```

## Developing with HMR
```
yarn run start
```

## Storybook
```
yarn run storybook
```
